# Walmart Weekly Sales — Regression Analysis

Exploratory data analysis (EDA) and baseline regression modeling on Walmart's weekly sales dataset. This project implements and compares a baseline Linear Regression model against a Random Forest Regressor on raw tabular features.

### Core Features:
* **Exploratory Data Analysis:** Includes multi-variable correlation analysis and holiday-effect data visualizations.
* **Predictive Modeling:** Compares foundational statistical modeling with ensemble tree-based machine learning approaches.
* **Evaluation Framework:** Quantifies performance using Mean Absolute Error (MAE), Mean Squared Error (MSE), and R-squared ($R^2$) metrics.

*Built as part of M.S. Data Science coursework.*
